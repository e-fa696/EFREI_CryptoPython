Fonctionnalités de cryptage

Utilisez les URLs suivantes pour tester les fonctionnalités :

  • Crypter une valeur : /encrypt/<valeur>
  • Décrypter une valeur : /decrypt/<valeur>
  • Définir une clé personnalisée : Envoyez une requête POST à /set-key avec un JSON contenant {"key": "votre_clé"}.
